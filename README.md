# Prism Labs AI — YouTube Companion Code

Companion code, examples, and resources for videos on the [Prism Labs AI](https://youtube.com/@prismlabsai) YouTube channel.

## Structure

```
Videos/
  yyyy-mm-dd_Video_Title/
    README.md          # Overview + setup instructions
    example_code.py    # Working code examples
    ...
```

Each video folder contains real, functional code that complements the video content. Clone any folder and follow the README to get started.

## Usage

```bash
# Clone the full repo
git clone https://github.com/PrismLabsAI/YouTube.git

# Or browse individual video folders on GitHub
```
